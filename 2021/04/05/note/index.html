笔记APP推荐

发表于 2021-04-05 更新于 2021-04-17 阅读次数：

本文总结了自己平时比较常用的几款软件，主要用于日常的文献阅读，做笔记和论文撰写

主要分以下四个场景：

  1. 文档类笔记：这类笔记的特点是文字多，有较为清晰的层次逻辑
  2. 手写笔记+OCR识别：这类笔记主要面向数学公式相关的笔记
  3. 云同步
  4. 提醒事项

推荐的软件有：Typora，GoodNotes，Mathnip，Onedrive，滴答清单

  1. 推荐使用Markdown进行笔记撰写，编辑器可用Typora，这款编辑器界面简洁，支持Mac/PC等多平台，此外在Mac平台还有一款熊掌记（Bear），UI十分漂亮，但是同步需要订阅且只支持苹果设备，为知笔记，印象笔记等也支持云同步和Markdown语法，功能十分完善，但也正因为如此，其复杂的功能反而导致学习成本的增加与日常使用的复杂
  2. 手写笔记：Ipad + Apple pencil，软件推荐GoodNotes（50块），可以通过iCloud等云同步，也可以导出PDF进行分享。嫌贵的小伙伴可以考虑OneNote，缺点是同步速度较慢，当我们手写完公式之后，配合Mathpix就可以将手写公式识别为latex语法，写论文时可以省很多功夫
  3. 如果你是学生，正好你的学校有微软教育服务，就可以去申请高达5T的Onedrive 存储空间，用来多设备文件同步，当然，也可以用坚果云代替，坚果云同步速度快
  4. 提醒事项推荐滴答清单，可以智能识别日期，配合apple watch ，多设备同步，体验非常好
